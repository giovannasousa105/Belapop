# Marketplace Luxo Checklist

## Brand and visual
- [ ] Black + pink + off-white only
- [ ] Typography hierarchy consistent
- [ ] High spacing density (editorial breathing room)

## Content quality
- [ ] No placeholders in public pages
- [ ] No "demo", "debug", "coming soon" copy
- [ ] Product and diary cards always route correctly

## Conversion quality
- [ ] Cart and checkout readable on mobile
- [ ] WhatsApp widget icon-only and non-intrusive
- [ ] CTA contrast and affordance validated

## Reliability
- [ ] Loading uses skeleton
- [ ] Empty and error states available
- [ ] No loading loops

## Security
- [ ] Admin routes hidden and protected
- [ ] Seller routes role-aware
- [ ] Supabase RLS validated
