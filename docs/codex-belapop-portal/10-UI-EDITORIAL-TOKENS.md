# UI Editorial Tokens (bp*)

## Palette
- `bpBlack`: `#0F0F10`
- `bpBlackSoft`: `#141416`
- `bpPink`: `#C2185B`
- `bpPinkSoft`: `#F4B6C2`
- `bpOffWhite`: `#FAF7F8`
- `bpGraphite`: `#2B2B2E`

## Radius
- `rounded-bpSm`: `10px`
- `rounded-bpMd`: `16px`
- `rounded-bpLg`: `22px`

## Shadows
- `shadow-bpMicro`
- `shadow-bpSoft`

## Rule
- Legacy style keys (`noir`, `blush`, `luxe`) must not be used in public UI.
- New work should only use `bp*` tokens.
