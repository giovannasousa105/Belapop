# CODEX VISUAL - Rosa & Preto (Editorial Clean Chic)

## Paleta (Refinada)
- Preto profundo: `#0F0F10`
- Preto suave (elevations): `#141416`
- Rosa queimado (acento principal): `#C2185B`
- Rosa blush (acento suave): `#F4B6C2`
- Off-white quente (fundo editorial): `#FAF7F8`
- Grafite texto secundario: `#2B2B2E`

## Regras
- Preto = base de luxo (header/footer/hero/CTA primario).
- Rosa = acento (links, highlights, badges discretos, bordas finas).
- Off-white = respiro editorial (paginas longas, grids, leitura).

## Tipografia (Direcao)
- Headline: serif editorial (Playfair Display / Cormorant Garamond).
- Texto/UI: sans clean (Inter / Satoshi).

## Espacamento e Layout
- Grid: 12 colunas (max width 1200-1280).
- Gutter: 24px desktop / 16px mobile.
- Muito espaco em branco: menos secoes e mais impacto por secao.
- Cards: borda fina + sombra quase imperceptivel.

## Fotografia
- 70% editorial / 30% funcional.
- Luz suave lateral, sombras elegantes.
- Evitar rosa neon saturado; preferir rosa queimado/rose.

## UI Patterns
- Botao primario: fundo preto, texto off-white, hover sutil.
- Botao secundario: outline preto, hover preenchimento leve.
- Links: rosa queimado.
- Badges: minimalistas (sem excesso de selos).

## O que NAO entra
- Carrossel com 6 banners de oferta no topo.
- Tags chamativas e coloridas.
- "Demo", "placeholder", "em breve" em vitrine publica.
- Loaders sem skeleton elegante.
